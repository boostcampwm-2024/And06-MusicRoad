## ✨ 프로젝트 소개
- 내가 Pick한 음악을 현재 위치에 기록할 수 있습니다. 
- 지도에서 나의 Pick뿐만 아니라 다른 사용자들의 Pick도 확인할 수 있습니다.
- 내 Pick을 공유하고 새로운 Pick을 발견하며 내 취향의 노래들을 모아 음악의 세계를 넓혀보세요.
